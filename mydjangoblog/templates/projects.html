Evolutionary Algorithm

University assignment

The algorithm is based on the biological evolutionary process. The algorithm starts with a random generation of individuals. Each individual has its fitness, which is the measure of how close It is to the desired result. Calculation of fitness varies from case to case. Once all the individuals are defined we assigned the fitness and sort the population according to it, then we apply the process of Mutation to the each individual but according to a certain probability. Mutation involves changing one or more values of a character. The new value must come from the legal set of values. After some of the individuals have mutated, some of the most fit individual are taken to produce new individuals that will be called children, this will be done using crossing-over with a certain probability. The crossover operation is a simple array copying, we take two parents at a time and a random index is generated, we will then use that index to divide the gene of the individuals in two parts. The first part of parent1 is combined with the second part of parent2 and vice versa, in this way we will produce two completely new individuals and with some probability most of them will have a greater fitness. To maintain the population size, we remove the worst individuals in the population and add the children. Of course this only happens if the children are not the worst individuals. The whole population will soon be replaced and all these steps are repeated until either the best individual is found or the number of times the algorithm ran on the population exceeds the given limit.

The best individual has the maximum fitness possible that varies according to the number of genes and this particular target is set by the user. Marks Obtained: 100/100

Digital Image Processor

Image Processing Tool

Image processing involves amending the color values of the pixels in an image in some way, or creating a new image from an old one having processed the old one in some way. In these project i created and applied filters to images. A filter is a 2D array of values and so are the images in question. The filter is usually a small array, for example 3 rows by 3 columns (3x3) but the image is much more larger. The values in the filter are used to modify each pixel in the source image simply applying matrix moltiplication.

Each single pixel in the source image is then modified by the entire filter.Filter used are Sepia,Greyscale,Identity,Blur,Sharpen,Edge,Emboss.
